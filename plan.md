# Project: Mack C App - WhatsApp Inspired Theme

## 1. Objective
Develop a messaging and video calling application with a visual identity closely inspired by WhatsApp's clean, functional, and recognizable interface.

## 2. Visual Theme (WhatsApp Inspired)
- **Primary Color**: WhatsApp Green (`#25D366` in Light, `#00A884` in Dark)
- **Secondary Colors**: Header Green (`#075E54`), Background Grey (`#F0F2F5`), Chat Bubbles (`#DCF8C6`).
- **Typography**: Clean sans-serif (Inter/system-ui).
- **Dark Mode**: Support for WhatsApp's "Deep Night" mode using `#111B21` background.

## 3. Key Features (Current Scope)
- **Theme Implementation**: Update global CSS variables to reflect WhatsApp colors.
- **Enhanced Landing Page**: 
    - Hero section showing a mock chat interface to demonstrate the theme.
    - WhatsApp-like navigation and footer.
- **User Authentication**:
    - Phone number-based login (Mock for now).
    - OTP verification screen.
- **Responsive Design**: Mobile-first approach, ensuring the app looks like the WhatsApp mobile app.

## 4. Technical Stack
- React 19 + Vite + Tailwind CSS
- Framer Motion for smooth transitions
- Lucide React for iconography
- Shadcn/UI for accessible components
